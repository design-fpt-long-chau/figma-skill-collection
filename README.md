# figma-skill-collection
A sources of reusable skills description for PO and Designers at Long Châu to use within Figma Agents

## How to use
1. Paste this link into Figma Plugins: https://raw.githubusercontent.com/design-fpt-long-chau/figma-skill-collection/main/skills.json
2. Select what skill you want to use within Figma
3. Use Figma Agent to execute the skill
